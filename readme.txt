== Description ==

wordpress plugin to retrieve message from frontlinesms software (http://www.frontlinesms.com) ,
this plugin will create a widget to show the message

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `wp-frontlinesms` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place widget to the bar or adding shortcode [wp-frontlinesms] to a Post or Page
